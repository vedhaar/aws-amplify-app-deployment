# AWS Amplify App Deployment

This project demonstrates deploying a web application using AWS Amplify.

## Project Overview
A web application was deployed and hosted using AWS Amplify, which provides continuous deployment and hosting for modern web applications.

## AWS Services Used
- AWS Amplify
- Git-based deployment

## Steps Performed
1. Created an app in AWS Amplify
2. Connected the repository
3. Configured build settings
4. Deployed the application
5. Accessed the app using the Amplify-generated URL
